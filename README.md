# aggr-v1-sc
Aggr smart contracts

**Requirements**

- nodeJS v16.2.0 or later
- npm 7.10.0 or later

**Installation**

- `yarn`

**Run tests**

- `yarn lint`
- `yarn test`
  
**Make Flattened contract file**

- `yarn flatten`

**Deploy**
- create you .env && .secret files (see .env.example, .secret.example)
- `yarn deploy --network rinkeby`
